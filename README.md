Веб-приложение на Spring Boot для генерации и хранения паролей. Поддерживает PostgreSQL, шифрует пароли (BCrypt).

Требования для запуска:
- Java 17
- Maven
- PostgreSQL
- Переменная `DB_PASSWORD`

1. Генерация пароля

   Запрос: `http://localhost:8080/api/password/generate?length={длина}&complexity={сложность}&owner={владелец}`
  - length – от 4 до 30
  - complexity – 1 (цифры), 2 (+буквы), 3 (+символы)
  - owner – имя (необязательно)

   Пример запроса :`http://localhost:8080/api/password/generate?length=12&complexity=3&owner=testUser`

3. **Сохранение, просмотр, обновление и удаление паролей**  
   (CRUD через REST API)

4. **Теги для паролей**  
   Можно создавать, добавлять и управлять тегами (например, "работа", "личное").

Как хранит данные:
- Пароли – зашифрованы в PostgreSQL
- Теги и связи – отдельные таблицы
